# ShopEase (Demo)

Minimal e-commerce product management demo built with Node.js, Express and SQLite.

Setup

Install dependencies and initialize the database:

```bash
npm install
npm run init-db
```

Run

```bash
npm start
# open http://localhost:3000
```

Features (minimal):
- Browse products
- Client-side cart (localStorage)
- Place orders (server-side)
- Basic admin endpoints to update inventory and promotions
